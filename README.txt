To add scripts:
	Right click on "Scripts"
	Choose "New Item"
	Ensure the folder selected is rundir\scripts.uosl\
	Choose "Utility" from the "Installed Templates" pane.
	Choose "Text File"
	Enter a name for the script with the file extension .uosl; i.e.: <filename>.uosl

For easier script editing in Visual Studio 10, check out the UOSL Language Package here:
	http://uodemo.joinuo.com/index.php?title=UOSL_Language_Package
